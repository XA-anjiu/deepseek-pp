## DeepSeek++ 0.5.0

0.5.0 是自动化与工具续跑稳定版本，重点提升长任务承接、历史展示清洁度和侧边栏加载体验。

### 主要变化

- 自动化任务完成后会保存更可靠的会话链接、父消息和历史快照，后续继续运行更稳。
- 自动化任务和手动 Agent 式续跑使用一致的工具执行与结果回传节奏，减少长任务中的状态差异。
- 会话历史和本地缓存会统一隐藏内部提示词与原始工具调用标记，同时保留可恢复的工具执行记录。
- 输出速度显示在流式回复和兼容请求路径中保持一致，减少速度状态残留。
- 侧边栏的记忆、Skill、预设、自动化、MCP 和设置页面改为按需加载，初次打开更轻。
- 新增提示词冻结检查，发布前确认关键系统提示、工具格式和续跑提示没有被无意改写。
- 将过期的活动规划文档归档为当前架构基线，避免后续工作继续引用旧重构计划。
