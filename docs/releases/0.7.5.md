## DeepSeek++ 0.7.5

0.7.5 是 Skill 本地工作流与 Agent 输出稳定性版本，让本机 Skill 更容易进入 DeepSeek++，同时让历史回复、自动化和 inline agent 收尾更干净。

### 主要变化

- **本地 Skill 管理**：Skill 页新增本机 Skill 预览、导入和同步能力，可把个人或团队本地 Skill 文件夹纳入 DeepSeek++ 工作流。
- **导入前可审查**：本地 Skill 导入前会展示名称、说明、来源和可用状态，帮助用户确认内容后再启用。
- **历史回复更干净**：恢复旧对话时，`task_complete` 工具块会显示为更自然的摘要，减少技术标记干扰。
- **自动化运行更稳**：PoW header 改为按需创建，降低自动化任务在请求准备阶段被打断的概率。
- **Inline agent 收尾减少重复**：任务已给出完成摘要时，inline agent 会复用该摘要，避免重复生成最终总结。
- **产品素材更新**：README 和 Chrome Web Store 素材换成新的产品截图，侧边栏、工具执行、自动化和项目能力展示更贴近当前界面。
- **回归覆盖补强**：新增本地 Skill 导入、同步合并、Shell Host 本地预览、历史清理、PoW、inline agent 和注入样式测试。
